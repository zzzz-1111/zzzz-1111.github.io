Sigmoid函数详解

为什么需要Sigmoid函数？

在二分类问题中，我们希望模型输出一个0到1之间的概率值，表示样本属于正类（y=1y=1y=1）的可能性。如果直接使用线性函数wTx+b\mathbf{w}^T\mathbf{x} + bwTx+b，输出可能是任意实数（比如-5或10），这无法表示概率。Sigmoid函数就像一个"软开关"，将任意实数平滑地映射到(0,1)区间。

Sigmoid函数定义：

σ(z)=11+e−z\sigma(z) = \frac{1}{1 + e^{-z}} σ(z)=1+e−z1​

其中z=wTx+bz = \mathbf{w}^T\mathbf{x} + bz=wTx+b是线性组合的结果。

Sigmoid函数的重要性质：

  • 当z→+∞z \to +\inftyz→+∞时，e−z→0e^{-z} \to 0e−z→0，所以σ(z)→1\sigma(z) \to 1σ(z)→1
  • 当z→−∞z \to -\inftyz→−∞时，e−z→+∞e^{-z} \to +\inftye−z→+∞，所以σ(z)→0\sigma(z) \to 0σ(z)→0
  • 当z=0z = 0z=0时，σ(0)=11+1=0.5\sigma(0) = \frac{1}{1+1} = 0.5σ(0)=1+11​=0.5
  • 函数图像呈S型，单调递增，输出严格在(0,1)之间

Sigmoid函数导数推导：

在反向传播中，我们需要计算Sigmoid函数的导数。让我们一步步推导：

设σ(z)=11+e−z=(1+e−z)−1\sigma(z) = \frac{1}{1+e^{-z}} = (1+e^{-z})^{-1}σ(z)=1+e−z1​=(1+e−z)−1

使用链式法则求导：

  • 外层函数：f(u)=u−1f(u) = u^{-1}f(u)=u−1，导数为f′(u)=−u−2f'(u) = -u^{-2}f′(u)=−u−2
  • 内层函数：u=1+e−zu = 1+e^{-z}u=1+e−z，导数为dudz=−e−z\frac{du}{dz} = -e^{-z}dzdu​=−e−z

应用链式法则：

dσ(z)dz=−1⋅(1+e−z)−2⋅(−e−z)=e−z(1+e−z)2\frac{d\sigma(z)}{dz} = -1 \cdot (1+e^{-z})^{-2} \cdot (-e^{-z}) = \frac{e^{-z}}{(1+e^{-z})^2} dzdσ(z)​=−1⋅(1+e−z)−2⋅(−e−z)=(1+e−z)2e−z​

现在将其转换为更有用的形式。注意到：

σ(z)=11+e−z,1−σ(z)=e−z1+e−z\sigma(z) = \frac{1}{1+e^{-z}}, \quad 1-\sigma(z) = \frac{e^{-z}}{1+e^{-z}} σ(z)=1+e−z1​,1−σ(z)=1+e−ze−z​

因此：

σ(z)(1−σ(z))=11+e−z⋅e−z1+e−z=e−z(1+e−z)2\sigma(z)(1-\sigma(z)) = \frac{1}{1+e^{-z}} \cdot \frac{e^{-z}}{1+e^{-z}} = \frac{e^{-z}}{(1+e^{-z})^2} σ(z)(1−σ(z))=1+e−z1​⋅1+e−ze−z​=(1+e−z)2e−z​

最终得到优雅的结果：

dσ(z)dz=σ(z)(1−σ(z))\boxed{\frac{d\sigma(z)}{dz} = \sigma(z)(1-\sigma(z))} dzdσ(z)​=σ(z)(1−σ(z))​

这个公式在后续梯度计算中非常重要！

最大似然估计（Maximum Likelihood Estimation）

什么是似然？

在推导损失函数之前，我们需要理解最大似然估计的思想。似然回答的问题是：给定一组参数，观察到当前数据的概率有多大？

想象你有一枚硬币，抛了10次得到7次正面。如果硬币是公平的（正面概率0.5），观察到这个结果的概率是多少？如果正面概率是0.7呢？最大似然估计就是要找到使观察数据出现概率最大的参数值。

Logistic回归的概率模型：

对于二分类问题，给定输入x(i)\mathbf{x}^{(i)}x(i)，我们的模型预测：

  • P(y(i)=1∣x(i))=y^(i)=σ(wTx(i)+b)P(y^{(i)}=1|\mathbf{x}^{(i)}) = \hat{y}^{(i)} = \sigma(\mathbf{w}^T\mathbf{x}^{(i)} + b)P(y(i)=1∣x(i))=y^​(i)=σ(wTx(i)+b)
  • P(y(i)=0∣x(i))=1−y^(i)P(y^{(i)}=0|\mathbf{x}^{(i)}) = 1 - \hat{y}^{(i)}P(y(i)=0∣x(i))=1−y^​(i)

我们可以用一个统一的公式表示：

P(y(i)∣x(i))=(y^(i))y(i)⋅(1−y^(i))1−y(i)P(y^{(i)}|\mathbf{x}^{(i)}) = (\hat{y}^{(i)})^{y^{(i)}} \cdot (1-\hat{y}^{(i)})^{1-y^{(i)}} P(y(i)∣x(i))=(y^​(i))y(i)⋅(1−y^​(i))1−y(i)

在这个公式中，指数上的yiy^iyi取值是只能是0或者1，所以其实这个式子起到了一个选择的作用。

验证这个公式：

  • 当y(i)=1y^{(i)} = 1y(i)=1时：P=(y^(i))1⋅(1−y^(i))0=y^(i)P = (\hat{y}^{(i)})^1 \cdot (1-\hat{y}^{(i)})^0 = \hat{y}^{(i)}P=(y^​(i))1⋅(1−y^​(i))0=y^​(i) ✓
  • 当y(i)=0y^{(i)} = 0y(i)=0时：P=(y^(i))0⋅(1−y^(i))1=1−y^(i)P = (\hat{y}^{(i)})^0 \cdot (1-\hat{y}^{(i)})^1 = 1-\hat{y}^{(i)}P=(y^​(i))0⋅(1−y^​(i))1=1−y^​(i) ✓

整个数据集的似然函数：

假设各样本相互独立，整个训练集的似然是各样本似然的乘积：

L(w,b)=∏i=1mP(y(i)∣x(i))=∏i=1m(y^(i))y(i)⋅(1−y^(i))1−y(i)L(\mathbf{w}, b) = \prod_{i=1}^{m} P(y^{(i)}|\mathbf{x}^{(i)}) = \prod_{i=1}^{m} (\hat{y}^{(i)})^{y^{(i)}} \cdot (1-\hat{y}^{(i)})^{1-y^{(i)}} L(w,b)=i=1∏m​P(y(i)∣x(i))=i=1∏m​(y^​(i))y(i)⋅(1−y^​(i))1−y(i)

对数似然：

为了方便计算（避免很多小数相乘），我们对似然函数取对数：

log⁡L(w,b)=∑i=1m[y(i)log⁡y^(i)+(1−y(i))log⁡(1−y^(i))]\log L(\mathbf{w}, b) = \sum_{i=1}^{m} [y^{(i)} \log \hat{y}^{(i)} + (1-y^{(i)}) \log(1-\hat{y}^{(i)})] logL(w,b)=i=1∑m​[y(i)logy^​(i)+(1−y(i))log(1−y^​(i))]

最大似然估计的目标就是找到参数w,b\mathbf{w}, bw,b使得log⁡L(w,b)\log L(\mathbf{w}, b)logL(w,b)最大。

损失函数推导

从最大似然到最小化损失：

在机器学习中，我们习惯于最小化损失函数而不是最大化似然函数。最大化对数似然等价于最小化负对数似然。

单样本损失函数：

ℓ(i)(y^(i),y(i))=−[y(i)log⁡y^(i)+(1−y(i))log⁡(1−y^(i))]\ell^{(i)}(\hat{y}^{(i)}, y^{(i)}) = -[y^{(i)} \log \hat{y}^{(i)} + (1-y^{(i)}) \log(1-\hat{y}^{(i)})] ℓ(i)(y^​(i),y(i))=−[y(i)logy^​(i)+(1−y(i))log(1−y^​(i))]

这就是著名的二元交叉熵损失（Binary Cross-Entropy Loss）。

交叉熵损失的直观理解：

让我们分析这个损失函数的行为：

当y(i)=1y^{(i)} = 1y(i)=1（真实标签为正类）时：

ℓ(i)=−log⁡y^(i)\ell^{(i)} = -\log \hat{y}^{(i)} ℓ(i)=−logy^​(i)

  • 如果y^(i)=1\hat{y}^{(i)} = 1y^​(i)=1（预测完全正确），则ℓ(i)=0\ell^{(i)} = 0ℓ(i)=0
  • 如果y^(i)→0\hat{y}^{(i)} \to 0y^​(i)→0（预测完全错误），则ℓ(i)→+∞\ell^{(i)} \to +\inftyℓ(i)→+∞

当y(i)=0y^{(i)} = 0y(i)=0（真实标签为负类）时：

ℓ(i)=−log⁡(1−y^(i))\ell^{(i)} = -\log(1-\hat{y}^{(i)}) ℓ(i)=−log(1−y^​(i))

  • 如果y^(i)=0\hat{y}^{(i)} = 0y^​(i)=0（预测完全正确），则ℓ(i)=0\ell^{(i)} = 0ℓ(i)=0
  • 如果y^(i)→1\hat{y}^{(i)} \to 1y^​(i)→1（预测完全错误），则ℓ(i)→+∞\ell^{(i)} \to +\inftyℓ(i)→+∞

这样，当预测正确时损失为0，预测错误时损失很大，这正是我们想要的！

整体成本函数：

对所有样本的损失取平均，得到整体成本函数：

J(w,b)=1m∑i=1mℓ(i)=−1m∑i=1m[y(i)log⁡y^(i)+(1−y(i))log⁡(1−y^(i))]\boxed{J(\mathbf{w}, b) = \frac{1}{m} \sum_{i=1}^{m} \ell^{(i)} = -\frac{1}{m} \sum_{i=1}^{m} [y^{(i)} \log \hat{y}^{(i)} + (1-y^{(i)}) \log(1-\hat{y}^{(i)})]} J(w,b)=m1​i=1∑m​ℓ(i)=−m1​i=1∑m​[y(i)logy^​(i)+(1−y(i))log(1−y^​(i))]​

我们的目标是找到w\mathbf{w}w和bbb，使得J(w,b)J(\mathbf{w}, b)J(w,b)最小。

梯度推导过程

为什么需要计算梯度？

要使用梯度下降法最小化损失函数，我们需要知道损失函数关于每个参数的变化率，即梯度。梯度告诉我们参数应该朝哪个方向调整以减小损失。

链式法则回顾：

在推导过程中会大量使用链式法则。如果y=f(u)y = f(u)y=f(u)且u=g(x)u = g(x)u=g(x)，那么：

dydx=dydu⋅dudx\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} dxdy​=dudy​⋅dxdu​

单样本梯度推导：

我们先推导单个样本的梯度，然后求和得到整体梯度。

设置中间变量：

  • z(i)=wTx(i)+bz^{(i)} = \mathbf{w}^T\mathbf{x}^{(i)} + bz(i)=wTx(i)+b
  • y^(i)=σ(z(i))\hat{y}^{(i)} = \sigma(z^{(i)})y^​(i)=σ(z(i))
  • ℓ(i)=−[y(i)log⁡y^(i)+(1−y(i))log⁡(1−y^(i))]\ell^{(i)} = -[y^{(i)} \log \hat{y}^{(i)} + (1-y^{(i)}) \log(1-\hat{y}^{(i)})]ℓ(i)=−[y(i)logy^​(i)+(1−y(i))log(1−y^​(i))]

步骤1：计算∂ℓ(i)∂y^(i)\frac{\partial \ell^{(i)}}{\partial \hat{y}^{(i)}}∂y^​(i)∂ℓ(i)​

∂ℓ(i)∂y^(i)=−[y(i)⋅1y^(i)+(1−y(i))⋅−11−y^(i)]\frac{\partial \ell^{(i)}}{\partial \hat{y}^{(i)}} = -\left[y^{(i)} \cdot \frac{1}{\hat{y}^{(i)}} + (1-y^{(i)}) \cdot \frac{-1}{1-\hat{y}^{(i)}}\right] ∂y^​(i)∂ℓ(i)​=−[y(i)⋅y^​(i)1​+(1−y(i))⋅1−y^​(i)−1​]

=−[y(i)y^(i)−1−y(i)1−y^(i)]= -\left[\frac{y^{(i)}}{\hat{y}^{(i)}} - \frac{1-y^{(i)}}{1-\hat{y}^{(i)}}\right] =−[y^​(i)y(i)​−1−y^​(i)1−y(i)​]

步骤2：计算∂y^(i)∂z(i)\frac{\partial \hat{y}^{(i)}}{\partial z^{(i)}}∂z(i)∂y^​(i)​

使用前面推导的Sigmoid导数：

∂y^(i)∂z(i)=y^(i)(1−y^(i))\frac{\partial \hat{y}^{(i)}}{\partial z^{(i)}} = \hat{y}^{(i)}(1-\hat{y}^{(i)}) ∂z(i)∂y^​(i)​=y^​(i)(1−y^​(i))

步骤3：应用链式法则计算∂ℓ(i)∂z(i)\frac{\partial \ell^{(i)}}{\partial z^{(i)}}∂z(i)∂ℓ(i)​

∂ℓ(i)∂z(i)=∂ℓ(i)∂y^(i)⋅∂y^(i)∂z(i)\frac{\partial \ell^{(i)}}{\partial z^{(i)}} = \frac{\partial \ell^{(i)}}{\partial \hat{y}^{(i)}} \cdot \frac{\partial \hat{y}^{(i)}}{\partial z^{(i)}} ∂z(i)∂ℓ(i)​=∂y^​(i)∂ℓ(i)​⋅∂z(i)∂y^​(i)​

=−[y(i)y^(i)−1−y(i)1−y^(i)]⋅y^(i)(1−y^(i))= -\left[\frac{y^{(i)}}{\hat{y}^{(i)}} - \frac{1-y^{(i)}}{1-\hat{y}^{(i)}}\right] \cdot \hat{y}^{(i)}(1-\hat{y}^{(i)}) =−[y^​(i)y(i)​−1−y^​(i)1−y(i)​]⋅y^​(i)(1−y^​(i))

展开并化简：

=−[y(i)(1−y^(i))−(1−y(i))y^(i)]= -[y^{(i)}(1-\hat{y}^{(i)}) - (1-y^{(i)})\hat{y}^{(i)}] =−[y(i)(1−y^​(i))−(1−y(i))y^​(i)]

=−[y(i)−y(i)y^(i)−y^(i)+y(i)y^(i)]= -[y^{(i)} - y^{(i)}\hat{y}^{(i)} - \hat{y}^{(i)} + y^{(i)}\hat{y}^{(i)}] =−[y(i)−y(i)y^​(i)−y^​(i)+y(i)y^​(i)]

=−[y(i)−y^(i)]= -[y^{(i)} - \hat{y}^{(i)}] =−[y(i)−y^​(i)]

得到重要结果：

∂ℓ(i)∂z(i)=y^(i)−y(i)\boxed{\frac{\partial \ell^{(i)}}{\partial z^{(i)}} = \hat{y}^{(i)} - y^{(i)}} ∂z(i)∂ℓ(i)​=y^​(i)−y(i)​

步骤4：计算关于参数的梯度

对权重w\mathbf{w}w：

∂ℓ(i)∂w=∂ℓ(i)∂z(i)⋅∂z(i)∂w=(y^(i)−y(i))x(i)\frac{\partial \ell^{(i)}}{\partial \mathbf{w}} = \frac{\partial \ell^{(i)}}{\partial z^{(i)}} \cdot \frac{\partial z^{(i)}}{\partial \mathbf{w}} = (\hat{y}^{(i)} - y^{(i)}) \mathbf{x}^{(i)} ∂w∂ℓ(i)​=∂z(i)∂ℓ(i)​⋅∂w∂z(i)​=(y^​(i)−y(i))x(i)

对偏置bbb：

∂ℓ(i)∂b=∂ℓ(i)∂z(i)⋅∂z(i)∂b=y^(i)−y(i)\frac{\partial \ell^{(i)}}{\partial b} = \frac{\partial \ell^{(i)}}{\partial z^{(i)}} \cdot \frac{\partial z^{(i)}}{\partial b} = \hat{y}^{(i)} - y^{(i)} ∂b∂ℓ(i)​=∂z(i)∂ℓ(i)​⋅∂b∂z(i)​=y^​(i)−y(i)

整体损失函数的梯度：

对于整体成本函数J(w,b)=1m∑i=1mℓ(i)J(\mathbf{w}, b) = \frac{1}{m} \sum_{i=1}^{m} \ell^{(i)}J(w,b)=m1​∑i=1m​ℓ(i)：

∂J∂w=1m∑i=1m(y^(i)−y(i))x(i)\boxed{\frac{\partial J}{\partial \mathbf{w}} = \frac{1}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)}) \mathbf{x}^{(i)}} ∂w∂J​=m1​i=1∑m​(y^​(i)−y(i))x(i)​

∂J∂b=1m∑i=1m(y^(i)−y(i))\boxed{\frac{\partial J}{\partial b} = \frac{1}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})} ∂b∂J​=m1​i=1∑m​(y^​(i)−y(i))​

梯度下降法

基本思想：

梯度下降法是一种迭代优化算法，其核心思想是：

  1. 计算当前参数下损失函数的梯度
  2. 沿着梯度的反方向更新参数（因为梯度指向函数增长最快的方向，我们要最小化函数）
  3. 重复直到收敛

参数更新公式：

w:=w−α∂J∂w\mathbf{w} := \mathbf{w} - \alpha \frac{\partial J}{\partial \mathbf{w}} w:=w−α∂w∂J​

b:=b−α∂J∂bb := b - \alpha \frac{\partial J}{\partial b} b:=b−α∂b∂J​

其中α\alphaα是学习率，控制每次更新的步长。符号":="表示赋值。

将我们推导的梯度代入：

w:=w−αm∑i=1m(y^(i)−y(i))x(i)\mathbf{w} := \mathbf{w} - \frac{\alpha}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)}) \mathbf{x}^{(i)} w:=w−mα​i=1∑m​(y^​(i)−y(i))x(i)

b:=b−αm∑i=1m(y^(i)−y(i))b := b - \frac{\alpha}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)}) b:=b−mα​i=1∑m​(y^​(i)−y(i))

学习率的选择：

学习率α\alphaα的选择很关键：

  • 太小：收敛很慢
  • 太大：可能震荡或发散
  • 常用值：0.01, 0.03, 0.1, 0.3, 1.0

梯度下降的变体：

  • 批量梯度下降：使用全部mmm个样本计算梯度
  • 随机梯度下降（SGD）：每次只用一个样本
  • 小批量梯度下降：每次用一小批样本（如64个）

完整训练算法

算法流程：

  1. 初始化参数：

    • w=0\mathbf{w} = \mathbf{0}w=0（或小随机值）
    • b=0b = 0b=0
    • 设置学习率α\alphaα和最大迭代次数

  2. 重复以下步骤直到收敛：

    a. 前向传播（Forward Propagation）：

    • 对每个样本iii，计算：

      z(i)=wTx(i)+bz^{(i)} = \mathbf{w}^T\mathbf{x}^{(i)} + b z(i)=wTx(i)+b

      y^(i)=σ(z(i))\hat{y}^{(i)} = \sigma(z^{(i)}) y^​(i)=σ(z(i))

    b. 计算成本：

    J=−1m∑i=1m[y(i)log⁡y^(i)+(1−y(i))log⁡(1−y^(i))]J = -\frac{1}{m} \sum_{i=1}^{m} [y^{(i)} \log \hat{y}^{(i)} + (1-y^{(i)}) \log(1-\hat{y}^{(i)})] J=−m1​i=1∑m​[y(i)logy^​(i)+(1−y(i))log(1−y^​(i))]

    c. 反向传播（Backward Propagation）：

    • 计算梯度：

      dw=1m∑i=1m(y^(i)−y(i))x(i)d\mathbf{w} = \frac{1}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)}) \mathbf{x}^{(i)} dw=m1​i=1∑m​(y^​(i)−y(i))x(i)

      db=1m∑i=1m(y^(i)−y(i))db = \frac{1}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)}) db=m1​i=1∑m​(y^​(i)−y(i))

    d. 更新参数：

    w:=w−α⋅dw\mathbf{w} := \mathbf{w} - \alpha \cdot d\mathbf{w} w:=w−α⋅dw

    b:=b−α⋅dbb := b - \alpha \cdot db b:=b−α⋅db

收敛判断：

算法在以下情况之一时停止：

  • 损失函数变化小于阈值：∣J(t)−J(t−1)∣<ϵ|J^{(t)} - J^{(t-1)}| < \epsilon∣J(t)−J(t−1)∣<ϵ
  • 梯度范数小于阈值：∣∣dw∣∣<ϵ||d\mathbf{w}|| < \epsilon∣∣dw∣∣<ϵ
  • 达到最大迭代次数

向量化实现示例：

# 向量化的前向传播
Z = np.dot(w.T, X) + b     # Z的形状是(1, m)
A = sigmoid(Z)             # A就是所有样本的预测值

# 向量化的梯度计算
dw = (1/m) * np.dot(X, (A - Y).T)  # dw的形状是(n_x, 1)
db = (1/m) * np.sum(A - Y)         # db是标量

# 参数更新
w = w - alpha * dw
b = b - alpha * db

补充——矩阵乘法

在上面的代码np.dot(w.T, x)中，np.dot就是python里内置的矩阵乘法

规则

如果A是(m,n)矩阵，B是(n, p)矩阵，则A*B的结果是一个(m, p)的矩阵（记为C）

则C中第i行第j列的结果，就是A矩阵第i行和B矩阵第j行的点积一行乘以一列

记忆："外维定形状，内维要相等"

  • 外维：第一个矩阵的行数 × 第二个矩阵的列数 = 结果形状

  • 内维：第一个矩阵的列数必须等于第二个矩阵的行数

常见模式

# 前向传播
(1, n) × (n, m) = (1, m)    # w.T × X

# 反向传播 - 权重梯度
(n, m) × (m, 1) = (n, 1)    # X × dZ.T

# 反向传播 - 上一层梯度
(n, m) × (m, n) = (n, n)    # 其他情况

乘法符号两端的两个数字是一样的（内维度）

转置

就是把矩阵的行和列交换，矩阵(m*n)变成矩阵(n*m)
