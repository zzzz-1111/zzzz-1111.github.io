题解 P4137【Rmq Problem / mex】

2025-12-02 题解

题目

Rmq Problem / mex

题目描述

有一个长度为 nnn 的数组 {a1,a2,…,an}\{a_1,a_2,\ldots,a_n\}{a1​,a2​,…,an​}。

mmm 次询问，每次询问一个区间内最小没有出现过的自然数。

思路

参考文章(题解 by Great_Influence)[https://www.luogu.com.cn/article/0drrer8j]

对于每个数字iii，我们设置它的权值p[i]p[i]p[i]为数字iii第一次出现的位置。

然后假设我们的左端点是1，对于右端点rrr，我们相当于要找到一个位置pospospos使得
∀i<pos,p[i]≤r\forall i < pos, p[i]\leq r∀i<pos,p[i]≤r，也就是找到最大的ttt使得max⁡i=0t−1≤r\max^{t-1}_{i=0}\le rmaxi=0t−1​≤r

我们可以通过二分一个t来确定。

然后对于左端点不唯一的情况。我们先把所有的询问离线，并按照左端点排序，这样我们就可以及时更新权值（倘若p[i]已经不在区间里了（p[i]<l），就把p[i]设置成i下一次出现的位置）这样保证p[i]要么在区间里，要么在区间的后面。如果下一次没有出现了，就设置到无穷大（表示在无穷大之后才出现）

我们在线段树上进行这个操作，观察到二分的t是p[i]p[i]p[i]的下标，作为index的，所以我们可以直接在线段树上进行这个二分操作，最后每次查询的时间复杂度就可以降低到O(log⁡n)O(\log n)O(logn)
