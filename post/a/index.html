Jupyter Lab和Pytorch的安装和使用

2025-12-02 AI

Jupyter Lab安装

使用pip安装

pip install jupyterlab

然后将python包的script文件夹添加到系统的PATH环境变量

包的位置可以使用命令pip show jupyterlab来查看。注意，是要将script文件夹而不是site-packages文件夹，exe文件才可以直接执行

然后在cmd敲入jupyter lab即可打开了。

修改默认工作目录

运行下面这行命令，将文件的目录记住，然后再打开对应的文件

jupyter notebook --generate-config

找到文件中的# c.ServerApp.notebook_dir = ''，去掉注释再将目录填进去就好了
改成类似c.ServerApp.notebook_dir = 'D:\\code\\jupyterlab'

Pytorch 安装

再官网链接上下载，然后安装。
anaconda下载链接

然后就可以在开始菜单启动Anaconda Prompt了

配置虚拟环境

conda create -n env_name python==version 
# 注：将env_name 替换成你的环境的名称，version替换成对应的版本号，eg：3.8

conda activate env_name
# 注：env_name 替换成对应的环境的名称
conda deactivate

注意，版本最好不要太高，太高会导致相关的库不适配

其他更详细的配置可以看这个教程：深入浅出PyTorch-PyTorch的安装
